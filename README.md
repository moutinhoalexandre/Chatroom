## CHATROOM

Chatroom is my first Chat Application using :
- [x] Node.js, 
- [x] Express, 
- [x] MongoDB, 
- [x] React.js, 
- [x] Socket.io,
- [x] JWT

## INSTALLATION
- [x]  Clone the repository

In a terminal > cd backend :
- [x] npm install,
- [x] nodemon server,

In a terminal > cd frontend :
- [x] npm install,
- [x] npm start,
- [x] enjoy ;)
